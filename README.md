Solução para gerenciamento de fluxo de caixa doméstico - Front-end

# Requisitos para execução da aplicação
Ter os seguintes programas instalados na máquina e realizado o(s) seguinte(s) procedimento(s):<br/>
- Git<br/>
- NodeJS a partir da versão 20<br/>
- Estar com a porta 5173 disponível para o servidor http desta aplicação<br/>
- Executar os passos de instalação e execução contidos no projeto de back-end desta solução - que podem ser encontrados através da seguinte url:<br/>
https://github.com/cristianomartinsdias82/gerenciamento-fluxo-caixa-domestico-lado-servidor/blob/main/README.md

# Preparação para execução da aplicação
Após ter os programas mencionados acima instalados:<br/>
- Abra um prompt de comando ou Powershell<br/>

- Baixe este repositório em algum diretório de sua preferência através do seguinte comando:<br/>
X:\caminho\para\projeto>git clone https://github.com/cristianomartinsdias82/gerenciamento-fluxo-caixa-domestico-lado-cliente.git [ENTER]<br/>

- Em seguida, navegue até a seguinte pasta através do seguinte comando:<br/>
cd X:\caminho\para\projeto [ENTER]<br/>

- No diretório, digite os seguintes comandos:<br/>
npm install [ENTER]<br/>
npm run dev [ENTER]<br/>
(Com este último comando, um navegador web devera se abrir apontando para o endereço do servidor web)
